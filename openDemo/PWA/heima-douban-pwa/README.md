# 传统页面升级为 PWA

## 简介

黑马 PWA-3小时带你实现渐进式WebAPP 豆瓣 demo 改 PWA 源码

## demo 说明

项目中涉及到一点 node 的知识，懂与不懂不影响对 demo 的阅读。
Web APP Manifest 需要在 https 协议 或者 http://localhost 下访问项目。也可以吧浏览器调到手机模式。

## 技术栈

HTML、CSS、JS、PWA

## 安装教程

下载 heima-douban-pwa

npm i
cd heima-douban-pwa
node app.js
